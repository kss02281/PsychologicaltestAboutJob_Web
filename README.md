# Occupational Psychological Test Website
## 직업심리검사 웹사이트

### **소개**
이 웹 사이트는 **28가지**의 문항을 통해 검사가 진행되며, 선택한 결과를 분석하여 본인이 직업을 선택할 때 어떤 가치를 중요하게 생각하는지 알려주고, 나와 맞는 직업을 추천해주는 웹 사이트 입니다.
